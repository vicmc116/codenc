  Enable Periodic REGISTRY Backup
Windows 10 and later versions have stopped backing up the registry periodically. 
Copy this registry key to reenable the periodic backups of the registry 
(backs up the registry every ten days). Copy the code below save it on your desktop in notepad, rename the extension from .txt to .reg and click… 
COPY ALL BELOW...
Windows Registry Editor Version 5.00[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Configuration Manager]"BackupCount"=dword:00000001"EnablePeriodicBackup"=dword:00000001
